INITIAL HARDWARE DETECTION - Basic GPU/CPU quality rating system.

Core Features:
1. getGpuQuality() - Returns 1-4 based on GPU name string matching
   - GeForce 8/9: Quality 1 (Low)
   - GeForce GTX 26x: Quality 2 (Medium)
   - GeForce GTX 2/4/5, AMD HD 4870 X2, HD 5800/5900, HD 69xx: Quality 3-4 (High/Ultra)
   - Multi-GPU bonus: Returns 4 if gpuCount > 1

2. getCpuQuality() - Formula: (cpuSpeed-1.2)*(cpuCount-1)
   - 2 cores @ 3.0 GHz = 1.8
   - 4 cores @ 3.2 GHz = 6.0
   - 8 cores @ 3.2 GHz = 14.0 (marked "awesome")

3. hardwareSpecs Array - 5 profiles indexed [0-4]
   - [0] = Auto-detect using detected hardware
   - [1-4] = Faked profiles for testing (GeForce 9, GTX 2, GTX 295, GTX 4xx)

Memory Specs (in bytes):
- Low: 1GB RAM, 512MB VRAM
- Medium: 1GB RAM, 768MB VRAM
- High: 2GB RAM, 1GB VRAM
- Ultra: 2GB RAM, 2GB VRAM

Limitations:
- No vendor database system
- No device ID lookup
- Simple string matching only
- No platform-specific handling
