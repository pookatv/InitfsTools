[description]
Game entry point - first payload loaded by the Frostbite engine.

[version]
2011-2016 (Battlefield, Battlefront, Garden Warfare, Need For Speed)
[notes]
Original Battlefield 3/4 era implementation.

Features:
- Basic platform detection only (Win32, PS3, Xenon)
- Simple logging setup
- Minimal global variables

Why it was simple: Games only shipped on 2-3 platforms max, and build configurations were handled elsewhere.
[content]
--
-- Game startup script
--

require "vfs"
core = require "Frost.Core"
logging = require "Frost.Logging"

-- The global platform variable can be used to detect which platform
-- we are currently executing on

platform = dice.getCurrentPlatformName()

-- Set up a logger which logs to the console (not stdout)
-- By default, we only show warnings or worse. For more verbose
-- diagnostics, you may set the level to INFO or DEBUG.

log = logging.ingameConsole()
log:setLevel(logging.DEBUG)

core.import("Scripts/Game/Configure.lua")
[/version]

[version]
2023 (Dead Space Remake)
[notes]
Added storage speed detection for next-gen optimization.

New Features:
- hardDriveType variable - Detects HDD vs SATA SSD vs NVMe SSD

Why: CRITICAL for PS5/Xbox Series X optimization - these consoles have ultra-fast NVMe drives.
- HDD = slow loading (legacy)
- SATA SSD = faster (mid-gen upgrades)
- NVMe SSD = instant (new gen mandatory installs)

Games adjusted streaming, LOD, and asset loading based on storage speed.

Comment: [DIVERGENCE] tag shows this was a platform-specific feature not in the main Frostbite branch yet.

Removed:
- platformSDK (Dead Space didn't need it)

[version]
2025+ (Battlefield 6, Skate, Future Titles)
[notes]
The complete 'gold standard' template for modern Frostbite games.

All major detection variables combined:
- config - Build configuration
- platform - Target platform
- platformMode - Client/server/editor
- platformSDK - SDK version
- hasClient - Graphical client present
- hardDriveType - Storage speed detection

This became the mainline Frostbite template going forward.

No comments - This is now mainline Frostbite, not a divergence.
