[description]
Frostbite engine pipeline configuration - controls asset compilation, shader generation, texture compression, and platform optimizations for the content build system.

[version]
2011 (Battlefield 3 Alpha)
[notes]
Early Frostbite 2 pipeline with PS3/Xbox 360 focus.

Key Features:
- Multi-platform DX10/10.1/11 shader paths
- PS3 EDGE geometry optimization (Cell SPU processing)
- CUDA-accelerated texture compression
- Basic Enlighten GI with radiosity
- Havok physics pipeline with validation
- Perforce integration for asset management

PS3-Specific Optimizations:
- Vertex/index buffer compression
- SPU-optimized mesh processing
- Edge geometry for deferred rendering on Cell
- Optimization level 3 shaders

Notable Limitations:
- No dynamic environment maps
- Basic terrain system
- Limited shader debugging
- Manual cache management
