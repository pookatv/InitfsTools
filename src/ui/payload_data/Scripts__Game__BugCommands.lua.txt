[description]
Bug reporting templates and QA debug commands - configures the in-game bug reporting system (B4Bug/BugDrone).

[version]
2011 (Battlefield 3)
[notes]
Original B4Bug implementation for Battlefield 3.

Features:
- StandardB4BugBug - Default bug template
- Basic JIRA field mapping (Issue Type, Priority, Assignee, Language)
- Player position tracking (soldier + camera positions)
- PC hardware info collection (OS, GPU, driver version)
- Changelist tracking (game + Frostbite versions)
- Juice Session ID integration (EA's telemetry system)
- Test commands for QA (rendering, animation, weapon debugging)

Why it was simple: Early Battlefield games focused on PC with basic bug tracking needs.
